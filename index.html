How to Install

Copy code below and paste it in the terminal.

Sublime Text 2

Installation

cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/
git clone https://github.com/nonlinear/codecode.git

Update

cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/codecode
git pull

Mixins macros

In order to use these macros, add mixins up somewhere on your SASS tree. HoverClick for more info.

  • mixins
  • box
  • centerer
  • clear
  • cover
  • devicewidth
  • disable
  • ellipsis
  • filter
  • font-face
  • grid
    • Simple grid with same-width columns that collapses automatically on phone devices.
    1. Apply it in the parent element
    2. Child elements (columns) should be either: li, dt, section or class .cell
    First variable is number of columns, second variable is gutter (in pixels).
    Requires mixins
  • gradient
  • grid
  • hide-text
  • media
    • Media queries for most devices. You can either use numbers, like
    @include media(200px, max) {
    }
    and it outputs as
    @media screen and (max-width: 200px) {
    }
    variables are number and min|max.
    Alternatively, you can use proper device names:
    • ipad
    • iphone6
    • iphone5
    • iphone4
    with more to come. scope variable options are strict (rules apply only to device), landscape (rules apply to landscape and smaller) and portrait (rules apply to portrait and smaller)
    Requires mixins
  • Only
    • only puts modernizr and detectizr to use. for instance,
    @include unless(touch) {
    background: red;
    }
    makes background red unless in touchscreen devices (but not on desktop), or
    @include unless(android) {
    display: none;
    }
    hides object only on android systems.
    Some useful Modernizr tags are
    • csscolumns
    • flexbox
    • fontface
    • touch
    • flash
    .
    Some useful Detectizr tags are systems
    • mac
    • windows
    • linux
    • android
    , browsers
    • ie
    • firefox
    • safari
    and browser versions
    • ie9
    • safari8
    .
    Requires mixins, modernizr, detectizr, SASS 3.1
  • opacity
  • orient
  • placeholder
  • radius
  • reflect
  • reset
    • Resets object properties. variables are
    • all (resets margin, padding, border)
    • margin (resets margin)
    • padding (resets padding)
    • border (resets border)
    • bg (resets background)
    • list (resets list)
    Requires mixins
  • rotate
  • scale
  • scroll-hint
  • shadow
  • size
  • skew
  • square
  • trans
  • translate
  • unless
    • unless puts modernizr and detectizr to use with rules that affect all instances except the one listed. for instance,
    @include unless(linux) {
    background: red;
    }
    makes background red everywhere, unless it's a linux machine.
    Some useful Modernizr tags are
    • csscolumns
    • flexbox
    • fontface
    • touch
    • flash
    .
    Some useful Detectizr tags are systems
    • mac
    • windows
    • linux
    • android
    , browsers
    • ie
    • firefox
    • safari
    and browser versions
    • ie9
    • safari8
    .
    Requires mixins, modernizr, detectizr, SASS 3.1
